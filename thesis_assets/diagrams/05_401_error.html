图5 无 Token 请求返回 401
$ curl -s -w "\nHTTP Status: %{http_code}" \ http://localhost:8080/api/service-b/inner/identity | jq
{ "code": 401, "message": "未授权，请先登录", "data": null } HTTP Status: 401
$ curl -s http://localhost:8080/api/service-b/inner/identity \ -H "Authorization: Bearer invalid-token" | jq
{ "code": 401, "message": "Token 无效或已过期", "data": null }
